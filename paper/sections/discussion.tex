\section{議論}
\label{sec:discussion}

\subsection{自己整合性評価への適用可能性}
\label{sec:disc:self_audit}

提案手法が持つ judge LLM 非依存かつ解釈可能な性質は、LLM 自身の出力を
評価する自己整合性ループに適用可能である。すなわち、ある LLM $M$ が
生成した応答 $t$ に対し、PoR 座標と $L_{\mathrm{sem}}$ の 3 項分解
を計算することで、$M$ は「自らの出力のどの誤差軸に問題があるか」を
診断信号として受け取れる。judge に別の LLM を要求する既存手法
(G-Eval や Prometheus 等) ではこのループを閉じる際に評価側モデルの
更新が診断の一貫性を損なうが、本手法は cascade matcher と SBert
のみで評価を完結するため、モデルバージョンに依存しない自己監査を
実装できる。これは LLM 開発過程での回帰検出や、実運用での応答
品質ゲートとして応用可能と考えられる。

\subsection{理論的位置付け}
\label{sec:disc:theoretical}

本研究で提案した 2 軸座標 PoR と損失関数 $L_{\mathrm{sem}}$ は、著者
らが提唱する意味場の構造モデル (Unconscious Gravity Hypothesis, UGH)
の枠組みに位置付けられる。UGH は応答テキストを意味空間上の語彙重力
分布として記述し、問いの核からの引力と構造的崩壊を定量化する理論的
動機を与えるが、本論文では評価指標としての実装と検証に焦点を絞る。
理論的背景の詳細は別稿に譲る。

\section{限界}
\label{sec:limitations}

本研究の限界を以下に列挙する。これらは大半が拡充版で対処可能である
が、年次大会版の主張範囲を正確に画定するため明示する。

\paragraph{サンプル規模.}
HA48 ($n = 48$) は予備検証規模であり、主指標 $\Delta E$ の Spearman
$\rho = +0.482$ は有意 ($p = 0.0005$) だが、95\% 信頼区間
$[+0.229, +0.674]$ は $\rho = 0.5$ を下端で下回る。また
ベースラインとの相関差は Steiger's $Z$ test で $\alpha = 0.05$
水準の有意差に到達しない。「相関強度 $\geq 0.5$」および「ベースラインを
有意に上回る」の 2 つの主張は本論文の scope 外であり、
$n \geq 100$ への拡張で確立することを今後の課題とする。

\paragraph{単一アノテータ.}
HA48 のアノテーションは単一アノテータによる作業であり、
inter-annotator agreement (Cohen's $\kappa$) は測定できていない。
参照上限 $\rho = +0.862$ (human $C$ 使用) は単一アノテータ前提下の
値であり、真の参照上限を過大評価している可能性がある。拡充版では
第二アノテータを導入して $\kappa$ を報告する予定である。

\paragraph{単一ドメイン.}
本研究の評価対象は AI 応答監査ドメインに限定される。要約品質評価、
対話応答評価、機械翻訳評価といった他ドメインへの汎化性は本論文では
検証していない。

\paragraph{ベースラインの範囲.}
比較は BLEU / BERTScore / SBert cosine の 3 系統に限定し、LLM 駆動
評価 (G-Eval, Prometheus, LLM-as-Judge) は judge LLM API 依存による
再現性問題から本論文では scope 外とした。これらとの直接比較は
今後の拡張課題である。

\paragraph{閾値の固定性.}
verdict 閾値 $0.10$ / $0.25$ は HA48 校正で確定した値をそのまま用い、
拡充版では再校正せず再現性の確認に用いる予定である。これにより
事後的なハイパーパラメータ調整 (p-hacking) の懸念を排除する一方、
拡充版での最適閾値が現行値から乖離した場合は別途追試を要する。
