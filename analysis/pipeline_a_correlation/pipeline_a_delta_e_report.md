## 3. 所見

### system C vs human C

- **ΔE_A (sys)**: ρ = -0.7737 ← メイン指標（デプロイ可能）
- **ΔE_A (human)**: ρ = -0.9266 ← 参照上限
- **C_sys**: ρ = 0.4030
- **C_human**: ρ = 0.9090 ← 参照上限

### ΔE_A (sys) vs cosine ΔE

- **ΔE_A (sys)**: ρ = -0.7737
- **delta_e_full** (cosine): ρ = -0.4283

ΔE_A (sys) は cosine ΔE より**強い**相関（|ρ| 0.7737 vs 0.4283）。

### ΔE_A (sys) vs C_sys 単独

- **C_sys**: ρ = 0.4030
- **1 - ΔE_A (sys)**: ρ = 0.7737

ΔE_A (sys) は C_sys 単独を**上回る**。S の統合が品質予測に寄与している。

### S（構造品質）の寄与

- **S**: ρ = 0.5770
- **1 - fail_max**: ρ = 0.5866

### Model C' との比較

- **Model C' quality_score**: ρ = 0.8292（既知参考値, t=0.0, system_hit_rate）
- **1 - ΔE_A (sys)**: ρ = 0.7737

Model C' (ρ=0.8292) が ΔE_A (sys) (|ρ|=0.7737) を上回る。

## 4. 結論

パイプラインAの理論式ΔEは S（構造品質）と C（命題カバレッジ）を加重二乗和で統合し、
理論的に整合的な距離指標を提供する。本検証でその実測相関が明らかになった。
