## 4. 設計判断への所見

### ΔE_A 単独 (Model 4) vs 現行 Model C' (Model 1)

現行 Model C' (ρ=0.6116) が ΔE_A 単独 (ρ=0.5179) を上回る。

### パラメータフィット (Model 3) vs ΔE_A 単独 (Model 4)

Model 3 (ρ=0.7872) が Model 4 (ρ=0.5179) を上回る。判定層のパラメータフィットに意義あり。

- Model 3 ρ_loo: 0.6446
