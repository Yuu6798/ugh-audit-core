# Model C' の L_R を ΔE_A に差し替えた場合の ρ 比較

**実行日**: 2026-04-02
**データ**: HA20 (20件, temperature=0.7)

## 0. データソースに関する重要な注記

本分析の `C`（命題カバレッジ）は `human_annotation_20_completed.csv` の `propositions_hit`
（人間アノテーション）から算出している。一方、既知の Model C' ρ=0.8292 は
`ha20_merged_for_model_c.csv` の `system_hit_rate`（システム自動判定）と異なる `fail_max` を
使用して算出されている。

主要な差分（human annotation vs system/merged data）:

| id | C (human) | system_hit_rate | fail_max (gate) | fail_max (merged) |
|----|-----------|-----------------|-----------------|-------------------|
| q032 | 0.000 | 1.000 | 0.0 | 1.0 |
| q075 | 1.000 | 0.667 | 1.0 | 0.0 |
| q044 | 0.333 | 0.667 | 1.0 | 0.0 |
| q100 | 0.667 | 0.333 | 0.0 | 0.5 |

そのため **Model 1 の ρ=0.7727 は既知の ρ=0.8292 を再現しない**。
5モデル間の比較は同一データソースを使用しているため公平だが、
既知の Model C' ρ=0.8292 との直接比較は**データソースが異なる**ことに留意が必要。

ΔE_A の高い ρ=0.9278 には、C が人間アノテーション由来（ρ=0.909 の内部一貫性を持つ）
であることが寄与している。system_hit_rate を使った場合の ΔE_A の ρ は未検証。
