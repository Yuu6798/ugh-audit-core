## 4. 設計判断への所見

### ΔE_A 単独 (Model 4) vs 現行 Model C' (Model 1)

現行 Model C' (ρ=0.6116) が ΔE_A 単独 (ρ=0.5179) を上回る。

### パラメータフィット (Model 3) vs ΔE_A 単独 (Model 4)

Model 3 (ρ=0.4892) が Model 4 (ρ=0.5179) を下回る。パラメータフィットが ΔE_A の予測力を劣化させている。

### LOO-CV ベースの比較

- Model 3 ρ_loo: 0.4763
- Model 4 ρ_loo: 0.5176

LOO-CV では Model 4 が優位。パラメータフィットは過学習の兆候がある。

### 総合判断

LOO-CV ベースで最良のモデルは **Model 1 (C'式, human C)** (ρ_loo=0.6108)。
